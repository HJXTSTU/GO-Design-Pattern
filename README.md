# GO-Design-Pattern
&emsp;&emsp;用Go把设计模式实现一遍。构建一个知识库。<br>
&emsp;&emsp;虽然网上有《Go Patterns》,不过是英文版的，并且不全，我就想把它翻译成中文的，并且补全。<br>

## 项目目录说明
* 顶级目录名为模式类别
* 二级目录名为具体的模式
* 三级目录名为贡献者名字
* 三级目录名为模式具体的说明文档以及实现代码
* 实现代码也可以是一个工程目录，尽量简洁

## 例如
* 创建型模式/单例模式/寒江雪/Singleton.md
* 创建型模式/单例模式/寒江雪/Singleton.go

## 提交说明
* 作者们自己新建一个分支
* 代码必须是go
